Créer vos propres champs pour vos articles/pages et proposer une recherche basée sur ces champs à vos visiteurs.

== Description ==

Boostez les champs personnalisés de Wordpress avec ce plugin:
- Création simplifiée et automatisée des champs lors de la rédaction de vos articles et pages. 
- Activez simplement le widget pour permettre aux visiteurs de votre site d'effectuer une recherche sur vos champs
- Customisez l'affiche du résultat de la recherche dans les options du plugin
- Insérez le shortcode dans votre contenu pour afficher les champs


== Installation ==

Pour installer le plugin: 
- Décompresser le fichier ZIP
- Placer le dossier décompréssé dans le dossier wp-content/plugins/ de votre site
- Dans l'administration de votre site, allez dans "Extensions" puis cliquer sur "Activer" pour le plugin "Recherche Multi-champs"
- Un nouveau menu "Recherche Multi-Champs" apparaîtra alors.
